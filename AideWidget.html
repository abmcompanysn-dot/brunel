Widget Mahu pour votre site web

Ajoutez un bouton flottant façon Tawk.to sur votre propre site : vos visiteurs vous laissent un message et une note, vous les recevez par email et dans votre Dashboard.

1. Récupérez votre code

Connectez-vous à votre Dashboard, ouvrez l'onglet Apparence & Liens, puis section "Widget pour votre site web". Votre code personnalisé y est déjà prêt, avec votre identifiant de profil intégré :

<script src="https://mahuprofile.abmcompanysn.workers.dev/widget.js?user=VOTRE-SLUG" async></script>

Cliquez sur "Copier le code" dans le Dashboard pour ne pas avoir à le retaper.

2. Collez-le sur votre site

01

Ouvrez le code source

Éditeur HTML, WordPress (thème > éditeur), Wix (code personnalisé), Shopify, etc.

02

Collez avant </body>

Le script doit être ajouté juste avant la balise de fermeture </body> de chaque page où vous voulez le widget.

03

Publiez

Le bouton flottant Mahu apparaît immédiatement en bas à droite de votre site.

3. Ce que voient vos visiteurs

En cliquant sur le bouton flottant, le visiteur peut :

  • Donner une note sur 5 étoiles à votre site.
  • Laisser son nom et un contact (email ou téléphone).
  • Écrire un message (optionnel).

💬

4. Ce que vous recevez

Chaque message envoyé depuis votre site apparaît immédiatement :

  • Dans votre boîte email, avec le nom, le contact, la note ⭐ et le message.
  • Dans votre Dashboard, section "Mes Prospects", avec le canal "🌐 Site Web" pour le distinguer des messages laissés directement sur votre profil Mahu.

Questions fréquentes

Le widget ralentit-il mon site ?

Non. Le script se charge en async et pèse quelques kilo-octets ; il ne bloque jamais l'affichage de votre page.

Puis-je le mettre sur plusieurs sites ?

Oui, le même code (avec votre identifiant) fonctionne sur autant de sites et de pages que vous voulez.

Puis-je changer les couleurs ou le texte ?

Pas encore depuis le Dashboard — le widget utilise actuellement un style par défaut aux couleurs Mahu. Contactez le support si vous avez un besoin spécifique.

Le widget fonctionne-t-il sans compte Mahu actif ?

Non, il vous faut un identifiant de profil valide (celui de votre Dashboard) pour que les messages soient acheminés jusqu'à vous.
