Foire Aux Questions

Comment fonctionne la carte Mahu ?
La carte Mahu utilise la technologie NFC. Il suffit de l'approcher d'un smartphone compatible pour partager instantanément votre profil numérique.
Ai-je besoin d'une application ?
Non, aucune application n'est nécessaire pour partager ou recevoir les informations. Tout se fait via le navigateur web du téléphone.
Puis-je modifier mes informations ?
Oui, vous pouvez mettre à jour votre profil à tout moment depuis votre tableau de bord. Les changements sont immédiats.
