5. Stratégie de cache Cloudflare

Durées de cache

ParamètreValeurEffet
s-maxage 86 400s (24h) Le profil est servi depuis Cloudflare sans appeler Google
max-age 3 600s (1h) Cache navigateur
stale-while-revalidate 604 800s (7 jours) Sert le cache périmé immédiatement + revalide en arrière-plan

Purge du cache après modification

Quand un profil est modifié (écriture), le Worker supprime automatiquement les entrées en cache pour les actions getProfileData et getPublicProfile du slug concerné. 

// Clé de cache construite :
/cache/getProfileData?u=jean-dupont
/cache/getPublicProfile?u=jean-dupont

// Purge immédiate après saveProfile / quickRegisterAndActivate
caches.default.delete(cacheKey)

7. Ce qu'il faut ajouter côté Google Apps Script

Important : Le frontend appelle maintenant quickRegisterAndActivate. Cette action doit être implémentée dans votre Google Apps Script pour traiter le formulaire d'activation complet.

Action à implémenter : quickRegisterAndActivate

Cette fonction reçoit un payload JSON et doit :

// Payload reçu depuis ActivationCarte.html :
{
  "slug": "jean-dupont",          // URL du profil
  "nom_complet": "Jean Dupont",
  "profession": "Directeur Commercial",
  "compagnie": "Acme Corp",
  "telephone": "+33 6 00 00 00 00",
  "email": "jean@acme.com",
  "password": "•••••••••",         // À hasher côté GAS
  "liens_sociaux": [              // Array des réseaux
    { "type": "linkedin", "url": "..." }
  ],
  "card_code": "MAHU-XXXX",       // Null si pas de code
  "lead_capture": "OUI"
}

// Réponse attendue si succès :
{ "success": true, "token": "..." }

// Réponse si erreur :
{ "success": false, "error": "Cet email est déjà utilisé" }

Étapes côté GAS :

  • Vérifier que le slug n'est pas déjà pris
  • Vérifier que l'email n'est pas déjà enregistré
  • Hasher le mot de passe (bcrypt ou SHA-256 avec sel)
  • Créer la ligne utilisateur dans Google Sheets
  • Créer la ligne profil (slug, nom, profession, liens…)
  • Si card_code fourni : lier la carte au profil dans la feuille Cards
  • Générer et retourner un token de session
